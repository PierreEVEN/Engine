

# VkEngine
## description

3D game engine prototype writen in C++ with Vulkan.


## features
- custom code reflection with auto-generated code using macros.
- async data loading
- custom gltf importer
- job system
- easy to use delegates
- class serialization using reflection
- shaders hot reload
- shader editor
